Masterbrand

Color Palette

The NFL Masterbrand color system. Eight carefully defined colors that anchor every NFL touchpoint across broadcast, digital, print, and out-of-home.

Primary Colors

Gridiron Black

HEX #000000
RGB 0 · 0 · 0
CMYK 0 · 0 · 0 · 100
Pantone Black C

The anchor of the NFL palette. Used for primary text, backgrounds, and wherever maximum contrast is required.

Sideline White

HEX #FFFFFF
RGB 255 · 255 · 255
CMYK 0 · 0 · 0 · 0

Clean and open. Used for body copy on dark backgrounds, reversed logos, and whitespace-driven layouts.

Blue Family

Legacy Blue

HEX #0B1829
RGB 11 · 24 · 41
Pantone 533 C

Deep, authoritative navy. Used as the darkest blue in gradient backgrounds and as a shadow tone.

NFL Shield Blue

HEX #003369
RGB 0 · 51 · 105
Pantone 654 C

The primary NFL blue — the color of the Shield. Dominant background color for brand expressions.

Impact Blue

HEX #0075FF
RGB 0 · 117 · 255
Pantone 285 C

Energetic and electric. Used for accents, CTA highlights, and digital interactive states.

Red Family

Legacy Red

HEX #4A0000
RGB 74 · 0 · 0
Pantone 7428 C

Deep heritage red. Paired with Legacy Blue in historical and heritage brand moments.

Varsity Red

HEX #C0190A
RGB 192 · 25 · 10
Pantone 186 C

Bold and competitive. The mid-tone red for most brand applications and partner activations.

NFL Shield Red

HEX #D50A0A
RGB 213 · 10 · 10
Pantone 186 C

The red of the Shield. Used for primary CTAs, key brand moments, and the NFL Shield itself.

Color Usage Ratios

NFL Shield Blue should dominate (60–70% of most layouts). Gridiron Black and Sideline White provide structure. NFL Shield Red is an accent — use it purposefully for maximum impact.

Reference: NFL Brand Book pages 14 and 24–25.

Do

Use NFL Shield Blue as your primary background. Pair white text on blue. Use Shield Red only for CTAs and key moments.

Do Not

Do not use Impact Blue as a background. Do not combine Varsity Red and Legacy Red in the same layout. Do not use off-brand colors.
